Overview

The Factsheets are a standardised collection and presentation of data in a format which emphasizes key points concisely. The use of interactive fields and pre-defined responses is designed to make it easy to insert and filter for existing modules. The Factsheets are seperated in three categories: Framework-, Model-, and Scenario- Factsheet.

When publishing a study, there is a full set of linked factsheets. Startig with a basic description of the used framework, an adapted model is described. Usually different scenarios with modified input data and assumptions are calculated and compared.
